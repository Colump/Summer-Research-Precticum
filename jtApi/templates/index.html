Journeyti.me - A Free Public Dublin Bus Journeytime Prediction API

The General Transit Feed Specification (GTFS) is a data specification that allows public transit agencies to publish their transit data in a format that can be consumed by a wide variety of software applications. See some of the incredible work done bringing this specification to life at gtfs.org. Today, the GTFS data format is used by thousands of public transport providers.

If you search for Journey Time Prediction applications for public transport systems in any large metro area and you are sure to find several solutions, many using advanced machine learning techniques tailored for best prediction accuracy in that area. At Journeyti.me, we wondered what might be possible if all the work done producing these disparate applications and their prediction models could be brought together, unified through something as simple as implementing a single API?

This idea gave birth to the Journeyti.me project. An early attempt at providing a documented, free, public API and service to provide Journey Time predictions. Initially, this API will only support journey predictions on Dublin Bus, in Dublin, Ireland. But the goal of the project is to provide a platform for standardised documentation and service provision. We hope to grow the specification to support more and more prediction models, providing a framework that would allow the excellent work done by localised teams to be leveraged using a standard implemenation scheme.
